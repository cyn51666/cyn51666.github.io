sql中的运算符

2020-10-28 SQL Server 笔记 0 评论

在sql语言中常用的运算符有以下几种：

  1. 算术运算符
    +（加）、-（减）、*（乘）、/（除）、%（取模，也就是商）

  2. 赋值运算符
    “=”（等号）表示赋值，例如：Name=‘王华’

  3. 比较运算符
    ==等于
    >大于
    <小于
    <>不等于
    >=大于等于
    <=小于等于
    !=不等于

  4. 逻辑运算符
    逻辑运算符用来对某个条件进行判断，以获得判断的真假，返回带有true或false值得布尔数据类型

    AND当且仅当两个布尔表达式都为true时，返回true

    OR当且仅当两个布尔表达式都为false时，返回false

    NOT对布尔表达式的值取反，优先级别最高
